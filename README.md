# 🎮 Rock Paper Scissors AI Bot

## 📌 Description
This project is an intelligent Rock-Paper-Scissors bot built in Python.  
Unlike random bots, this bot analyzes opponent patterns and predicts future moves to gain an advantage.

## 🚀 Features
- Tracks opponent move history
- Detects repeating patterns
- Predicts next move
- Uses counter strategy to win

## 🧠 How It Works
The bot:
1. Stores opponent moves
2. Looks for repeating sequences (patterns)
3. Predicts the opponent’s next move
4. Plays the winning counter move

## 📊 Performance
- Wins 60%+ against all provided bots
- Adapts to different strategies

## 🛠️ Technologies Used
- Python
- Basic AI logic (pattern recognition)
